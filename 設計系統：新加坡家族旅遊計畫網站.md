# 設計系統：新加坡家族旅遊計畫網站

## 概述

本設計系統由 UI/UX Pro Max 為新加坡家族旅遊計畫網站生成，旨在優化用戶體驗、提升視覺吸引力和可訪問性。

## 模式 (Pattern)

**名稱**：故事驅動 + Hero 模式  
**CTA 位置**：首屏上方  
**頁面結構**：Hero > 功能 > CTA

### 設計理念

故事驅動模式非常適合旅遊網站，因為它：
- 讓用戶沉浸在旅遊故事中
- 通過視覺敘述激發旅遊欲望
- 建立情感連接
- 鼓勵用戶參與和分享

## 風格 (Style)

**名稱**：Motion-Driven（動作驅動）  
**關鍵詞**：動畫豐富、微互動、平滑過渡、滾動效果、視差、入場動畫、頁面過渡

### 特性

- **性能**：⚠ 良好
- **可訪問性**：⚠ 尊重 prefers-reduced-motion
- **最適合**：作品集網站、故事講述平台、互動體驗、娛樂應用、創意、SaaS

### 實現方式

- 使用 Intersection Observer 實現滾動動畫
- 懸停效果過渡時間 300-400ms
- 頁面入場動畫
- 視差效果（3-5 層）
- 頁面過渡動畫

| 角色 | 十六進制 | 用途 | RGB |
|------|---------|------|-----|
| 主要色 | #0EA5E9 | 主要品牌色、標題、重點 | rgb(14, 165, 233) |
| 次要色 | #38BDF8 | 次要元素、強調 | rgb(56, 189, 248) |
| CTA 色 | #F97316 | 按鈕、行動呼籲 | rgb(249, 115, 22) |

| 文字色 | #0C4A6E | 主要文本 | rgb(12, 74, 110) |

### 色彩理念

**天空藍 + 冒險橙**

- **天空藍**（#0EA5E9）：象徵自由、探險、天空和海洋，適合旅遊主題
- **冒險橙**（#F97316）：象徵能量、熱情和冒險，鼓勵用戶採取行動

### 色彩應用

- **導航欄**：天空藍背景，白色文字
- **按鈕**：冒險橙背景，白色文字
- **卡片**：淺藍背景（#F0F9FF），天空藍邊框
- **文本**：深藍色（#0C4A6E）確保可讀性
- **強調**：次要藍色（#38BDF8）用於次要信息

## 關鍵效果 (Key Effects)

### 1. 滾動動畫 (Scroll Animation)

**技術**：Intersection Observer API

```javascript
// 當元素進入視口時觸發動畫
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
    }
  });
});
```

**效果**：
- 日期卡片從下往上滑入
- 景點標籤逐個出現
- 時間軸項目依次顯示

### 2. 懸停效果 (Hover States)

**過渡時間**：300-400ms

```css
.card {
  transition: all 300ms ease-in-out;
}

.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 12px 24px rgba(14, 165, 233, 0.2);
}
```

**效果**：
- 卡片向上浮起
- 陰影加深
- 邊框顏色變化

### 3. 入場動畫 (Entrance Animation)

**應用**：
- 頁面加載時的淡入效果
- 導航欄從上往下滑入
- 內容區域逐個出現

### 4. 視差效果 (Parallax)

**層數**：3-5 層

```css
/* 背景層移動速度較慢 */
.parallax-bg {
  transform: translateY(calc(var(--scroll) * 0.5px));
}

/* 前景層移動速度較快 */
.parallax-fg {
  transform: translateY(calc(var(--scroll) * 0.8px));
}
```

### 5. 頁面過渡 (Page Transitions)

**效果**：
- 頁面切換時的淡入淡出
- 平滑的內容替換
- 無縫的用戶體驗

## 要避免的反模式 (Anti-patterns)

❌ **通用照片** - 使用特定的旅遊目的地照片  
❌ **複雜預訂流程** - 簡化預訂步驟  
❌ **過度動畫** - 保持動畫簡潔，尊重 prefers-reduced-motion  
❌ **低對比度** - 確保文本對比度至少 4.5:1  
❌ **非響應式設計** - 支持所有設備尺寸

## 實現建議

### 1. 色彩實現

```html
<!-- CSS 變量定義 -->
<style>
  :root {
    --color-primary: #0EA5E9;
    --color-secondary: #38BDF8;
    --color-cta: #F97316;
    --color-bg: #F0F9FF;
    --color-text: #0C4A6E;
  }
</style>
```

### 2. 排版實現

```css
/* 標題樣式 */
h1 {
  font-family: 'Noto Sans Thai', sans-serif;
  font-size: 48px;
  font-weight: 700;
  line-height: 1.2;
  color: var(--color-primary);
}

/* 正文樣式 */
p {
  font-family: 'Noto Sans Thai', sans-serif;
  font-size: 16px;
  font-weight: 400;
  line-height: 1.6;
  color: var(--color-text);
}
```

### 3. 動畫實現

```css
/* 滾動進入動畫 */
.animate-in {
  animation: slideUp 0.6s ease-out forwards;
}

@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 懸停效果 */
.card {
  transition: all 300ms ease-in-out;
}

.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 12px 24px rgba(14, 165, 233, 0.2);
}
```

### 4. 響應式設計

```css
/* 移動設備 (375px) */
@media (max-width: 767px) {
  h1 { font-size: 32px; }
  p { font-size: 14px; }
}

/* 平板設備 (768px) */
@media (min-width: 768px) and (max-width: 1023px) {
  h1 { font-size: 40px; }
  p { font-size: 15px; }
}

/* 桌面設備 (1024px+) */
@media (min-width: 1024px) {
  h1 { font-size: 48px; }
  p { font-size: 16px; }
}
```
