# Accessibility Checklist for Multi-Generational Trips

Use this checklist to ensure your itinerary is accessible and enjoyable for all family members.

## For Elderly Participants

### Physical Accessibility
- [ ] Include walking distance estimates for each activity
- [ ] Note locations with stairs, elevators, or ramps
- [ ] Suggest rest periods between activities
- [ ] Identify accessible restroom locations
- [ ] Recommend comfortable walking shoes
- [ ] Include seating availability information

### Health Considerations
- [ ] List nearby medical facilities and pharmacies
- [ ] Include emergency contact numbers
- [ ] Note air-conditioned rest areas
- [ ] Suggest hydration breaks in hot climates
- [ ] Include dietary accommodation options
- [ ] Provide information about wheelchair/mobility device rental

### Transportation
- [ ] Recommend accessible transportation options
- [ ] Include taxi/ride-sharing app information
- [ ] Note public transit accessibility
- [ ] Suggest direct routes vs. scenic routes
- [ ] Include estimated travel times

## For Children and Families

### Activity Suitability
- [ ] Age recommendations for activities
- [ ] Estimated duration for each activity
- [ ] Bathroom break locations
- [ ] Feeding/snack time recommendations
- [ ] Nap-friendly activities or rest periods
- [ ] Child-friendly restaurant options

### Safety
- [ ] Emergency contact procedures
- [ ] Lost child protocol
- [ ] Safe meeting points at each location
- [ ] Sun protection recommendations
- [ ] Water safety information
- [ ] Allergy/dietary restriction accommodations

### Comfort
- [ ] Stroller/carrier accessibility
- [ ] Changing table locations
- [ ] Nursing room availability
- [ ] Temperature considerations
- [ ] Noise level expectations
- [ ] Crowd level predictions

## For All Participants

### Communication
- [ ] Provide information in multiple languages (Chinese, English)
- [ ] Use clear, simple language
- [ ] Include visual maps and directions
- [ ] Provide contact numbers for all locations
- [ ] Share itinerary in multiple formats (HTML, PDF, print)

### Flexibility
- [ ] Build in buffer time between activities
- [ ] Offer alternative activities for different interests
- [ ] Suggest optional activities
- [ ] Include "free time" for rest or spontaneous activities
- [ ] Provide backup indoor activities for bad weather

### Information Accessibility
- [ ] Use large, readable fonts (minimum 12pt)
- [ ] High contrast between text and background
- [ ] Avoid excessive use of colors alone to convey information
- [ ] Include both text and icons
- [ ] Provide text descriptions for images
- [ ] Use simple, clear headings

## Document Accessibility

### Digital Format
- [ ] HTML version is mobile-friendly
- [ ] PDF is readable on all devices
- [ ] Print version is clear and legible
- [ ] Links are descriptive and functional
- [ ] Images have alt text descriptions

### Print Format
- [ ] Font size is at least 12pt
- [ ] High contrast (black on white preferred)
- [ ] Clear page breaks
- [ ] Readable on both sides of paper
- [ ] Includes page numbers
- [ ] Table of contents provided

## Pre-Trip Communication

### Send Information Early
- [ ] Share itinerary at least 2-3 weeks in advance
- [ ] Include all important details and contact information
- [ ] Request special accommodation needs
- [ ] Collect dietary restrictions and allergies
- [ ] Ask about mobility or health concerns
- [ ] Provide emergency contact procedures

### Confirm Understanding
- [ ] Follow up with participants to confirm receipt
- [ ] Ask if anyone has questions or concerns
- [ ] Offer to clarify any confusing information
- [ ] Provide alternative contact methods (email, phone, messaging)

## On-Site Accessibility

### Day-of Adjustments
- [ ] Be prepared to modify schedule if needed
- [ ] Have backup activities ready
- [ ] Monitor participants for fatigue or discomfort
- [ ] Adjust pace based on group capability
- [ ] Provide frequent rest breaks
- [ ] Stay flexible with timing

### Communication
- [ ] Confirm meeting times and locations
- [ ] Provide real-time updates if plans change
- [ ] Keep emergency contacts readily available
- [ ] Maintain regular check-ins with all participants
- [ ] Have backup transportation arranged

## Specific Accessibility Features

### For Mobility Issues
- [ ] Wheelchair accessibility information
- [ ] Parking availability and proximity
- [ ] Elevator/ramp locations
- [ ] Accessible restroom locations
- [ ] Seating availability
- [ ] Walking distance alternatives

### For Vision Impairment
- [ ] Large print versions available
- [ ] High contrast materials
- [ ] Audio descriptions of locations
- [ ] Tactile maps if possible
- [ ] Digital versions compatible with screen readers

### For Hearing Impairment
- [ ] Written information for all audio presentations
- [ ] Visual alerts and notifications
- [ ] Captioning for videos
- [ ] Alternative communication methods
- [ ] Note about sign language availability

### For Dietary Restrictions
- [ ] Vegetarian/vegan options at all meals
- [ ] Allergen information for restaurants
- [ ] Halal/Kosher options if needed
- [ ] Gluten-free alternatives
- [ ] Local markets for special dietary needs
- [ ] Restaurant reservation notes about restrictions

## Post-Trip Feedback

### Gather Feedback
- [ ] Ask about accessibility challenges faced
- [ ] Request suggestions for improvements
- [ ] Collect testimonials about positive experiences
- [ ] Use feedback to improve future itineraries
- [ ] Share improvements with group

### Documentation
- [ ] Update accessibility notes based on feedback
- [ ] Create accessibility guide for future trips
- [ ] Document successful accommodations
- [ ] Share best practices with other organizers
